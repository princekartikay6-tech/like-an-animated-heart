# like-an-animated-heart
Animated heart project exploring smooth motion and UI animation principles.

i) An interactive animated heart built using modern web animation techniques. 
ii) A smooth UI animation experiment focused on timing, motion, and visual feedback.  
iii) Creative motion design showcasing animation-driven storytelling.

Technical Angle

Built using HTML, CSS, JavaScript (GSAP-style animation logic) to achieve smooth and controlled motion.
Explores animation curves, easing, and frame-perfect transitions.
